# MERN Search Application

## Overview

This is a MERN stack application that allows users to search for courses stored in a MongoDB database based on their title and description.

## Features

- **Search Interface**: Users can input search criteria to find courses.
- **Backend API**: Node.js and Express.js server handles search requests and course creation.
- **Database**: MongoDB is used to store course data.
- **Deployment**: Application can be hosted on a suitable hosting platform.

## Technologies Used

- **Frontend**: React.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **HTTP Client**: Axios
- **Styling**: CSS (for basic styling)
